# astuneon.co — Astu Neon holding company site

Next.js (App Router) site for Astu Neon, Inc. Visionary narrative leads; consumer
brands (olive oil, platform, studio, consulting) live on their own domains.

Content source of truth: `Site Content & Decisions.md`.

## Develop

```bash
npm install
npm run dev
```
